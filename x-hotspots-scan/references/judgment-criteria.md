# X Hotspots Scan - Judgment Criteria

## What to Keep

### High-Value Signals
- **Concrete data**: "10x normal output", "98% context savings", "678 KB binary"
- **Specific projects**: GitHub repos, new tool launches, major version releases
- **Actionable workflows**: How someone is actually using AI/agents to get work done
- **High engagement**: Indicates the content resonates (controversial or valuable)
- **KOL posts**: People known for quality insights (levelsio, trq212, karpathy, etc.)

### Low-Value Noise (Skip)
- Pure rants/emotion without substance
- Ads/promos (unless it's a major product launch from a key player)
- Duplicate content (same story from multiple sources - keep the best one)
- Off-topic: Pure politics/entertainment/sports unless it's a global breaking news event
- "Me too" replies without new information

## Quality Bar
Ask yourself: "Would I want to read this in a curated daily digest?" If yes, keep it. If not, skip it.

## Priority Order
1. 🔴 High: KOL posts with high engagement + concrete data
2. 🟡 Medium: Interesting projects, workflow examples, moderate engagement
3. 🟢 Low: Only include if there's extra space and it's still somewhat relevant
